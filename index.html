What is BharatDroid

A phone-side operator for real Android tasks.

BharatDroid is an open-source Android app that acts like an AI operator for your phone. Instead of giving you suggestions and making you finish the work manually, it reads your intent, studies the live screen, and carries out the steps directly on-device.

The goal is simple: if you can say it naturally, your phone should be able to do it. Food delivery, messaging, media playback, payments, or routine app navigation should feel closer to asking a capable assistant than poking through menus.

The vision

Build an AI that can use the phone the way people do.

BharatDroid is aiming for a future where your Android device can handle chained actions across apps, without shipping your private behavior to a remote control center.

"Book the cab. Message my wife. Put on some music."

One request, multiple actions, one consistent agent that can plan, execute, and show proof of what it did. That is the shape of the product BharatDroid is growing toward.

The problem

Phones are powerful, but the interface is still painfully manual.

AI talks, but usually stops there

Most assistants can answer or recommend, but they leave the tapping, typing, and app-hopping to you.

Mobile tasks break across apps

Real life actions spill across food apps, chat, payments, transport, and media. Existing assistants rarely connect them.

Control should stay with the device

A useful assistant needs context and speed, but users should not have to surrender every action to a cloud dashboard.

The solution

An intent-first system that plans, executes, and confirms.

BharatDroid combines an AI brain, a screen-aware agent, and Android accessibility controls so the phone can understand the UI it is looking at and act with purpose.

Onboarding

How to get BharatDroid working from zero.

The flow is straightforward: create your Telegram bot, collect the token, add it inside the app, enable the Android permissions, and then start messaging your phone.

What you need

Four things before the first run

  • An Android phone running Android 11 or newer
  • A Telegram account
  • A bot token created with Telegram BotFather
  • An AI API key to power the agent inside the app
01

Create the Telegram bot

Open Telegram, message @BotFather, send /newbot, choose a name, and finish with a username that ends in bot.

02

Copy the bot token

BotFather gives you a token string after creation. Keep that safe because BharatDroid uses it to receive and send your commands through Telegram.

03

Add the token and AI key inside the app

Open BharatDroid on the phone, paste the Telegram bot token, paste your AI provider key, and save the setup so the agent can start understanding requests.

04

Enable Accessibility Service

This is the core permission. Turn on BharatDroid in Android accessibility settings so it can tap, type, scroll, and navigate the screen for you.

05

Enable notification access if you want relays

Optional but useful: notification access lets BharatDroid forward incoming app notifications to Telegram and help you reply remotely.

06

Start the agent and send the first task

Once the permissions are on, start the agent and message your bot in Telegram with a real request like searching, playing media, or opening and using an app.
