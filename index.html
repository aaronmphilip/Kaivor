Not another chatbot

It is a phone operator for the small jobs you keep postponing.

The taste test is simple: if the task needs opening apps, reading a screen, typing details, checking a result, or asking you one missing question, BharatClaw should be able to run the loop instead of making you tap through it.

The brain

Plan. See. Tap. Recover.

BharatClaw is useful only if it can survive real app screens. The agent chooses a skill when the task is known, then uses screen context and Android actions to keep moving when the UI changes.

Plan

Breaks the Telegram request into a runnable phone task.

See

Reads buttons, fields, lists, warnings, and current app state.

Act

Taps, types, scrolls, opens apps, and checks what changed.

Recover

Asks you for missing info instead of hallucinating details.

Visible progress

The notch is the receipt that the phone is working.

Running, waiting, continuing, stopped, or done should be visible on the phone. If overlay permission is missing, the foreground notification still carries the task state.
