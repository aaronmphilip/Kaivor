How it works

Fast path first. Visual reading when the screen needs it.

Kaivor starts with the Android accessibility tree because it is fast. It turns on visual screen reading for sparse screens, documents, form recovery, and anything the accessibility layer cannot explain.

  • Progress appears in the notch overlay and notification.
  • Gemini powers reasoning and document understanding.
  • Voice notes become commands through speech-to-text.
  • Blocked tasks ask for missing information in Telegram.

Product principle

Quiet interface. Relentless phone execution.

One command, visible state, exact questions when details are missing, and a final report when the Android task is done.
