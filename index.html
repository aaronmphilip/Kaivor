How it works

Fast path first. Visual reading when the screen needs it.

BharatClaw starts with the Android accessibility tree because it is fast. It turns on visual screen reading for forms, documents, sparse screens, and stuck recovery.

  • Progress is shown in the notch and notification.
  • Voice notes become commands through STT.
  • Missing information comes back to Telegram.
