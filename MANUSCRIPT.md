# BharatClaw Manuscript

## Vision

Build a WhatsApp-first AI growth + sales system for India that turns conversations into business outcomes.

Never miss a WhatsApp lead again, and convert more of them automatically.

## What BharatClaw is

- A system that replies
- Captures leads
- Follows up
- Helps close

## What BharatClaw is not

- Not a chatbot demo
- Not a dashboard tool
- Not an AI toy

## Long-term evolution

- Auto-reply -> lead capture
- Lead capture -> follow-ups
- Follow-ups -> conversions
- Conversions -> automation loops
- Automation loops -> autonomous agents

Final state: businesses run parts of operations through WhatsApp without hiring more people.

## Product thesis

Indian businesses do not need more software. They need fewer missed opportunities.

So we focus on:

- Faster replies
- Structured conversations
- Automatic follow-ups
- Simple conversion flows

## Core behavior

System should always:

- Reply instantly
- Ask useful questions
- Capture data
- Follow up automatically
- Notify owner
- Allow human takeover anytime

## Product principles

1. Simplicity: QR -> connect -> works
2. Language-first: natural Hinglish/English by default
3. No dashboard dependency: value stays inside WhatsApp
4. Outcome-driven: every feature should capture, convert, or save time

## Monetization philosophy

People do not pay for AI.
They pay for saved leads, faster replies, and better conversions.

## North star

Leads captured + follow-ups executed

## MVP spec

Core modules:

- WhatsApp connector
- State machine
- Reply engine
- Storage
- Scheduler
- Notifications
- Config

Core MVP features:

- WhatsApp connection (QR)
- Incoming message detection
- Auto-reply
- Lead capture flow
- Follow-up system
- Owner notification
- Manual takeover
- Basic config

## MVP lead flow logic

New message ->
Send greeting ->
Ask name ->
Ask requirement ->
Store data ->
Notify owner ->
Trigger follow-up if silent
